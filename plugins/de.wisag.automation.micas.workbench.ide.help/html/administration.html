BENDE - Baustein bedingtes Task-Ende

Übersicht

Funktion

Wenn am Eingang "Ende" 1-Signal anliegt, dann wird die Task beendet.
Liegt dagegen am Eingang "Ende" 0-Signal, so wird dieser Baustein übersprungen, und alle nachfolgenden Bausteine werden normal durchlaufen.

ENDE - Baustein Task-Ende

Übersicht

Funktion

Der ENDE-Baustein definiert das Ende eines MICAS-Programms. Er muß stets als letzter Baustein einer Task aufgerufen werden. Er muß zusammen mit einem TASK-Baustein verwendet werden, der vorher aufgerufen wurde.

Symbol

Funktion

Der Baustein hat keine Ein-/Ausgänge.
